## О книге

Книга написана с нуля для разработчиков которые уже умеют программировать, но не знают SolidJS. Нет "объяснений для чайников" — сразу к делу, с реальными примерами которые можно использовать в production.

**Что внутри:**
- 23 части, 133 раздела
- 11 живых примеров на 150-300 строк (автокомплит, Kanban, infinite scroll, форма регистрации, toast система, тёмная тема, data table с URL state, файловый загрузчик и др.)
- ASCII-диаграммы реактивного графа, SSR streaming, For vs Index
- Часть XIX — SolidStart от нуля до деплоя (12 разделов)
- Часть XXII — шпаргалки API, UnoCSS, частые ошибки, глоссарий
- Часть XXIII — 4 дополнительных production-примера

## Содержание (кратко)

| Часть | Тема |
|-------|------|
| I–II | Введение, JS/TS для Python разработчиков |
| III | Реактивность: сигналы, эффекты, мемо, реактивный граф |
| IV–VI | Компоненты, управление потоком, lifecycle |
| VII–IX | Stores, Context, Async / Suspense |
| X–XI | Роутинг, Формы (все типы полей, валидация, Zod) |
| XII–XIII | Работа с DOM, интеграции, TypeScript глубоко |
| XIV–XVIII | Продвинутые темы, стилизация, тестирование, debugging, производительность |
| XIX | SolidStart: SSR, server functions, actions, deploy |
| XX–XXI | Экосистема (Kobalte, TanStack), анти-паттерны |
| XXII–XXIII | Финальный проект, справочники, дополнительные примеры |

## Стек в примерах

- **SolidJS 1.9** + **TypeScript**
- **UnoCSS** (presetWind3) для стилей
- **Zod** для валидации форм
- **SolidStart** для SSR/полноценных приложений
- **Prisma** в финальном проекте
