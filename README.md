**Что это значит на практике:**

## О книге

Практическое руководство по SolidJS для разработчиков которые уже умеют программировать, но не знают SolidJS. Без воды — сразу к делу, с примерами которые можно использовать в production.

**Что внутри:**
- 23 части, 133 раздела, ~400 KB текста
- 11 живых примеров на 150–300 строк каждый
- ASCII-диаграммы реактивного графа, SSR streaming, For vs Index
- Часть XIX — SolidStart от нуля до деплоя (12 разделов)
- Шпаргалки API, UnoCSS, частые ошибки, глоссарий

## Содержание (кратко)

| Часть | Тема |
|-------|------|
| I–II | Введение, JS/TS для Python разработчиков |
| III | Реактивность: сигналы, эффекты, мемо, реактивный граф под капотом |
| IV–VI | Компоненты, управление потоком, lifecycle |
| VII–IX | Stores, Context, Async / Suspense |
| X–XI | Роутинг, Формы (все типы полей, валидация, Zod) |
| XII–XIII | Работа с DOM, интеграции с библиотеками, TypeScript глубоко |
| XIV–XVIII | Продвинутые темы, стилизация, тестирование, debugging, производительность |
| XIX | SolidStart: SSR, server functions, actions, cookies, deploy |
| XX–XXI | Экосистема (Kobalte, TanStack), анти-паттерны |
| XXII–XXIII | Финальный проект-блог, справочники, дополнительные практические примеры |

## Стек в примерах

- **SolidJS 1.9** + **TypeScript**
- **UnoCSS** (presetWind3) для стилей
- **Zod** для валидации форм
- **SolidStart** для SSR и полноценных приложений
- **Prisma** в финальном проекте-блоге
